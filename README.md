# Front-end Locasys

É uma aplicação front-end baseado no back-end do Locasys com intuíto de agilizar o controle de equipamentos local.
Permite visualizar equipamentos que se encontra no local, com quem está esse equipamento,
permite cadastro de filial, setor, usuário e fornecedor.
Sistema contém dashbord de equipamento, usuário, setor, fornecedor etc.. e cada um tem seu relátorio. 

## Tecnologias
- Next.js
- TailwindCSS


## Instalação

```bash
git clone https://github.com/Rodrigueness7/Front-end-LocaSys.git

cd projeto

npm install
```

## Configuração

Crie variável de ambiente na raiz do projeto:

#### .env.local

```bash
NEXT_PUBLIC_LOCALHOST = 127.0.0.1
```

## Como executar

```bash
npm run dev
```
